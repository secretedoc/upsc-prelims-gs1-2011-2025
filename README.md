# UPSC CSE Prelims - Raw PYQ Data

### About
This repository contains raw text files of Previous Year Questions (PYQs) for the UPSC Civil Services Preliminary Examination. 

### Purpose
This data is currently in an unstructured `.txt` format. It is being collected and will be processed into a structured database (JSON/CSV) to power a dedicated PYQ practice platform for UPSC aspirants.
